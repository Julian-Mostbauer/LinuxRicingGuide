Gentoo History

Gentoo Linux is a Linux distribution built using the Portage package management system. Unlike a binary software distribution, the source code is compiled locally according to the user's preferences and is often optimized for the specific type of computer.

  • 2000 – The Beginning

    Gentoo Linux was created by Daniel Robbins in 2000. It was initially named Enoch Linux before being renamed to Gentoo Linux.

  • 2002 – Gentoo 1.0

    Gentoo 1.0 was released in March 2002, featuring the Portage package management system and a focus on performance and customization.

  • 2004 – Gentoo 2004.0

    Gentoo 2004.0 was released in March 2004, featuring various improvements to the system's performance and usability.

  • 2008 – Gentoo 2008.0

    Gentoo 2008.0 was released in July 2008, featuring the Linux 2.6.24 kernel and various updates to the system's software packages.

  • 2011 – Gentoo 11.0

    Gentoo 11.0 was released in March 2011, featuring the Linux 2.6.37 kernel and various improvements to the system's performance and stability.

  • 2014 – Gentoo 20140826

    Gentoo 20140826 was released in August 2014, featuring the Linux 3.14.14 kernel and various updates to the system's software packages.

  • 2019 – Gentoo 20190826

    Gentoo 20190826 was released in August 2019, featuring the Linux 4.19.57 kernel and various improvements to the system's performance, stability, and security.

  • 2023 – Gentoo 20230826

    Gentoo 20230826 was released in August 2023, featuring the Linux 5.15.57 kernel and various updates to the system's software packages.
