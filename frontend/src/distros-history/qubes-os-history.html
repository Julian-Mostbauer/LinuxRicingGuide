Qubes OS History

Qubes OS is a security-focused desktop operating system that aims to provide security through isolation. Virtualization is performed by Xen, and user environments can be based on Fedora, Debian, Whonix, and Microsoft Windows, among other operating systems.

  • 2012 – The Beginning

    Qubes OS was first released in September 2012. It was created by Invisible Things Lab, a security research and consulting company.

  • 2014 – Qubes OS 2

    Qubes OS 2 was released in September 2014, introducing various improvements to the system's security and usability.

  • 2016 – Qubes OS 3.2

    Qubes OS 3.2 was released in October 2016, featuring significant enhancements to the system's performance and security.

  • 2018 – Qubes OS 4.0

    Qubes OS 4.0 was released in March 2018, introducing a new architecture that further improves the system's security and flexibility.

  • 2020 – Qubes OS 4.0.3

    Qubes OS 4.0.3 was released in February 2020, providing various updates and improvements to the system's stability and security.

  • 2021 – Qubes OS 4.1

    Qubes OS 4.1 was released in August 2021, featuring numerous enhancements to the system's performance, usability, and security.
