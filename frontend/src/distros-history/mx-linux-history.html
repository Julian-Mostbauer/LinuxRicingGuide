MX Linux History

MX Linux is a cooperative venture between the antiX and former MEPIS communities, using the best tools and talents from each of these distributions. It is designed to combine an elegant and efficient desktop with simple configuration, high stability, solid performance, and medium-sized footprint.

  • 2014 – The Beginning

    MX Linux was first released in March 2014 as a collaborative project between the antiX and MEPIS communities, aiming to provide a user-friendly and efficient desktop experience.

  • 2016 – MX-15

    MX-15 was released in December 2015, featuring the Xfce desktop environment and various improvements to the system's performance and usability.

  • 2017 – MX-16

    MX-16 was released in December 2016, introducing new features and enhancements to the system's stability and user experience.

  • 2018 – MX-17

    MX-17 was released in December 2017, featuring the Xfce 4.12 desktop environment and various updates to the system's software packages.

  • 2019 – MX-18

    MX-18 was released in December 2018, providing numerous improvements to the system's performance, stability, and security.

  • 2020 – MX-19

    MX-19 was released in October 2019, featuring the Xfce 4.14 desktop environment and various enhancements to the system's usability and functionality.

  • 2021 – MX-21

    MX-21 was released in October 2021, featuring the Xfce 4.16 desktop environment and numerous updates to the system's software packages and performance.

  • 2023 – MX-23

    MX-23 was released in August 2023, providing various improvements to the system's performance, stability, and security.
