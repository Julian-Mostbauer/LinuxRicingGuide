Pop!_OS History

Pop!_OS is a Linux distribution developed by System76, based on Ubuntu. It is designed for STEM and creative professionals who use their computer as a tool to discover and create. Pop!_OS provides full out-of-the-box support for both AMD and NVIDIA GPUs.

  • 2017 – The Beginning

    Pop!_OS was first released in October 2017, based on Ubuntu 17.10. It was developed by System76 to provide a better experience for their hardware.

  • 2018 – Pop!_OS 18.04 LTS

    Pop!_OS 18.04 LTS was released in April 2018, based on Ubuntu 18.04 LTS. This release included various improvements and new features tailored for System76 hardware.

  • 2019 – Pop!_OS 19.04

    Pop!_OS 19.04 was released in April 2019, featuring new features such as the Pop!_Shop, a custom app store, and improved support for hybrid graphics.

  • 2020 – Pop!_OS 20.04 LTS

    Pop!_OS 20.04 LTS was released in April 2020, based on Ubuntu 20.04 LTS. This release introduced new features like auto-tiling and improved support for flatpak applications.

  • 2021 – Pop!_OS 21.04

    Pop!_OS 21.04 was released in June 2021, featuring the new COSMIC desktop environment, which provides a more customizable and user-friendly experience.

  • 2022 – Pop!_OS 22.04 LTS

    Pop!_OS 22.04 LTS was released in April 2022, based on Ubuntu 22.04 LTS. This release included various performance improvements and new features to enhance the user experience.

  • 2023 – Pop!_OS 23.04

    Pop!_OS 23.04 continued the tradition of providing a user-friendly and powerful Linux distribution with the latest software and features.
