Linux Mint History

Linux Mint is a community-driven Linux distribution based on Ubuntu and Debian that strives to be a modern, elegant, and comfortable operating system which is both powerful and easy to use. It is one of the most popular desktop Linux distributions and used by millions of people.

Year-by-Year Summary of Linux Mint's History

  • 2006 – The Beginning

    Linux Mint was first released in August 2006 by Clement Lefebvre. The initial release was based on Kubuntu and included a number of multimedia codecs that were not included in Ubuntu by default.

  • 2008 – Linux Mint 5 "Elyssa"

    Linux Mint 5 "Elyssa" was released in June 2008, based on Ubuntu 8.04 LTS. This release included long-term support and various improvements to the user interface and system stability.

  • 2010 – Linux Mint 9 "Isadora"

    Linux Mint 9 "Isadora" was released in May 2010, based on Ubuntu 10.04 LTS. This release introduced the new Software Manager and various enhancements to the Update Manager.

  • 2012 – Linux Mint 13 "Maya"

    Linux Mint 13 "Maya" was released in May 2012, based on Ubuntu 12.04 LTS. This release featured the MATE and Cinnamon desktop environments, providing users with a choice of traditional or modern interfaces.

  • 2014 – Linux Mint 17 "Qiana"

    Linux Mint 17 "Qiana" was released in May 2014, based on Ubuntu 14.04 LTS. This release marked the beginning of a new strategy, with all future versions based on the same LTS release to ensure stability and long-term support.

  • 2016 – Linux Mint 18 "Sarah"

    Linux Mint 18 "Sarah" was released in June 2016, based on Ubuntu 16.04 LTS. This release included updates to the Cinnamon and MATE desktop environments and various improvements to the system's performance and usability.

  • 2018 – Linux Mint 19 "Tara"

    Linux Mint 19 "Tara" was released in June 2018, based on Ubuntu 18.04 LTS. This release introduced the new Timeshift backup tool and various enhancements to the Update Manager and Software Manager.

  • 2020 – Linux Mint 20 "Ulyana"

    Linux Mint 20 "Ulyana" was released in June 2020, based on Ubuntu 20.04 LTS. This release included improvements to the Cinnamon desktop environment, better support for NVIDIA GPUs, and various performance enhancements.

  • 2023 – Linux Mint 21 "Vanessa"

    Linux Mint 21 "Vanessa" was released in August 2023, based on Ubuntu 22.04 LTS. This release continued the tradition of providing a stable and user-friendly desktop experience with the latest software and features.
