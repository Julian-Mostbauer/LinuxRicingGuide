Elementary OS History

Elementary OS is a Linux distribution based on Ubuntu. It is known for its focus on user experience, design, and simplicity. Elementary OS aims to provide a polished and cohesive desktop environment with its custom Pantheon desktop.

Year-by-Year Summary of Elementary OS's History

  • 2011 – The Beginning

    Elementary OS was first released in March 2011, codenamed "Jupiter". It was based on Ubuntu 10.10 and introduced the Pantheon desktop environment.

  • 2013 – Elementary OS 0.2 "Luna"

    Elementary OS 0.2 "Luna" was released in August 2013, based on Ubuntu 12.04 LTS. This release included significant improvements to the Pantheon desktop and various custom applications.

  • 2015 – Elementary OS 0.3 "Freya"

    Elementary OS 0.3 "Freya" was released in April 2015, based on Ubuntu 14.04 LTS. This release brought new features, improved performance, and enhanced user experience.

  • 2017 – Elementary OS 0.4 "Loki"

    Elementary OS 0.4 "Loki" was released in September 2016, based on Ubuntu 16.04 LTS. This release introduced new applications, improved system settings, and various performance enhancements.

  • 2018 – Elementary OS 5.0 "Juno"

    Elementary OS 5.0 "Juno" was released in October 2018, based on Ubuntu 18.04 LTS. This release included a refreshed user interface, new features, and various improvements to the overall system.

  • 2020 – Elementary OS 5.1 "Hera"

    Elementary OS 5.1 "Hera" was released in December 2019, based on Ubuntu 18.04 LTS. This release brought new features, improved performance, and enhanced user experience.

  • 2021 – Elementary OS 6.0 "Odin"

    Elementary OS 6.0 "Odin" was released in August 2021, based on Ubuntu 20.04 LTS. This release introduced a new installer, improved system settings, and various performance enhancements.

  • 2023 – Elementary OS 7.0 "Horus"

    Elementary OS 7.0 "Horus" was released in January 2023, based on Ubuntu 22.04 LTS. This release continued the tradition of providing a polished and cohesive desktop experience with the latest software and features.
