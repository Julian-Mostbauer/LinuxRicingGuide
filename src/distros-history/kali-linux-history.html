Kali Linux History

Kali Linux is a Debian-derived Linux distribution designed for digital forensics and penetration testing. It is maintained and funded by Offensive Security Ltd. It was developed by Mati Aharoni and Devon Kearns of Offensive Security through the rewrite of BackTrack, their previous forensics Linux distribution.

Year-by-Year Summary of Kali Linux's History

  • 2013 – The Beginning

    Kali Linux was first released on March 13, 2013, as a complete rebuild of BackTrack Linux, adhering strictly to Debian development standards.

  • 2015 – Kali Linux 2.0

    Kali Linux 2.0, codenamed "Kali Sana," was released on August 11, 2015, featuring a new user interface and updated penetration testing tools.

  • 2016 – Rolling Release

    In 2016, Kali Linux transitioned to a rolling release model, providing continuous updates and improvements to the system.

  • 2019 – Kali Linux 2019.4

    Kali Linux 2019.4 was released in November 2019, introducing the new "Kali Undercover" mode, which makes the Kali desktop look like a default Windows installation.

  • 2020 – Kali Linux 2020.1

    Kali Linux 2020.1 was released in January 2020, featuring a new default non-root user policy and various updates to the system's tools and performance.

  • 2021 – Kali Linux 2021.1

    Kali Linux 2021.1 was released in February 2021, providing numerous improvements to the system's performance, stability, and security tools.

  • 2023 – Kali Linux 2023.1

    Kali Linux 2023.1 was released in March 2023, featuring various updates and enhancements to the system's penetration testing tools and overall performance.
