Artix Linux History

Artix Linux is a rolling-release distribution, based on Arch Linux. It uses OpenRC, runit, and s6 init instead of systemd. Artix Linux offers a lightweight, rolling-release operating system featuring the OpenRC init software.

Year-by-Year Summary of Artix Linux's History

  • 2017 – The Beginning

    Artix Linux was first released in 2017, providing a systemd-free Arch-based distribution with a focus on simplicity and user control.

  • 2018 – Artix Linux 2018.01.01

    Artix Linux 2018.01.01 was released in January 2018, featuring various improvements to the system's performance and init systems.

  • 2019 – Artix Linux 2019.06.01
