Parrot Security OS History

Parrot Security OS is a Linux distribution based on Debian with a focus on computer security. It is designed for penetration testing, vulnerability assessment, and mitigation, computer forensics, and anonymous web browsing.

  • 2013 – The Beginning

    Parrot Security OS was first released in 2013, aiming to provide a comprehensive suite of tools for security professionals and enthusiasts.

  • 2015 – Parrot Security OS 2.0

    Parrot Security OS 2.0 was released in 2015, featuring various improvements to the system's performance and security tools.

  • 2017 – Parrot Security OS 3.0

    Parrot Security OS 3.0 was released in 2017, introducing new features and enhancements to the system's stability and user experience.

  • 2019 – Parrot Security OS 4.0

    Parrot Security OS 4.0 was released in 2019, providing numerous improvements to the system's performance, stability, and security tools.

  • 2021 – Parrot Security OS 4.11

    Parrot Security OS 4.11 was released in 2021, featuring various updates to the system's software packages and performance.

  • 2023 – Parrot Security OS 5.0

    Parrot Security OS 5.0 was released in 2023, providing various improvements to the system's performance, stability, and security tools.
