Linux Distributions

Ubuntu

Ubuntu is a free and open-source Linux distribution based on Debian. Ubuntu is officially released in three editions: Desktop, Server, and Core for Internet of things devices and robots. All the editions can run on the computer alone, or in a virtual machine.

For more information, visit the Ubuntu website.

Debian

Debian is a Unix-like operating system consisting entirely of free software. Ian Murdock started the Debian Project on August 16, 1993. Debian 0.01 was released on September 15, 1993, and the first stable version, 1.1, was released on June 17, 1996.

For more information, visit the Debian website.

Arch Linux

Arch Linux is a Linux distribution for computers with x86-64 processors. Arch Linux adheres to five principles: simplicity, modernity, pragmatism, user centrality, and versatility. In practice, this means the project attempts to have minimal distribution-specific changes, minimal breakage with updates, pragmatic over ideological design choices, user-friendliness, and minimal bloat.

For more information, visit the Arch Linux website.

Manjaro

Manjaro is a free and open-source Linux distribution based on the Arch Linux

For more information, visit the Manjaro website.

Pop!_OS

Pop!_OS is a Linux distribution developed by System76, based on Ubuntu. It is designed to have minimal clutter on the desktop without a lot of bloat. The distribution is aimed at people who want a fast, stable, and clean system.

For more information, visit the Pop!_OS website.

Linux Mint

Linux Mint is a community-driven Linux distribution based on Ubuntu or Debian. It is designed to be user-friendly and comes with a variety of open-source applications. Linux Mint is the 4th most popular desktop operating system in the world.

For more information, visit the Linux Mint website.

Elementary OS

Elementary OS is a Linux distribution based on Ubuntu. It is designed to be easy to use and visually appealing. The distribution is aimed at people who want a fast, stable, and clean system.

For more information, visit the Elementary OS website.

CentOS

CentOS is a Linux distribution that provides a free, community-supported computing platform functionally compatible with its upstream source, Red Hat Enterprise Linux (RHEL). In January 2014, CentOS announced the official joining with Red Hat while staying independent from RHEL, under a new CentOS governing board.

For more information, visit the CentOS website.

Fedora

Fedora is a Linux distribution developed by the community-supported Fedora Project which is sponsored primarily by Red Hat, a subsidiary of IBM, with additional support from other companies. Fedora contains software distributed under various free and open-source licenses and aims to be on the leading edge of free technologies.

For more information, visit the Fedora website.

openSUSE

openSUSE is a Linux distribution sponsored by SUSE Software Solutions Germany GmbH and other companies. It is widely used throughout the world. The focus of its development is creating usable open-source tools for software developers and system administrators, while providing a user-friendly desktop and feature-rich server environment.

For more information, visit the openSUSE website.

Slackware

Slackware is a Linux distribution created by Patrick Volkerding in 1993. Originally based on Softlanding Linux System, Slackware has been the basis for many other Linux distributions, most notably the first versions of SUSE Linux distributions, and is the oldest distribution that is still maintained.

For more information, visit the Slackware website.

Gentoo

Gentoo Linux is a Linux distribution built using the Portage package management system. Unlike a binary software distribution, the source code is compiled locally according to the user's preferences and is often optimized for the specific type of computer.

For more information, visit the Gentoo website.

Alpine Linux

Alpine Linux is a Linux distribution based on musl and BusyBox, designed for security, simplicity, and resource efficiency. It uses a hardened kernel and compiles all user-space binaries as position-independent executables with stack-smashing protection.

For more information, visit the Alpine Linux website.

MX Linux

MX Linux is a midweight Linux operating system based on Debian stable and using core antiX components, with additional software created or packaged by the MX community.

For more information, visit the MX Linux website.

Ubuntu Studio

Ubuntu Studio is a recognized flavor of the Ubuntu Linux distribution, which is explicitly geared to general multimedia production. The original version, based on Ubuntu 7.04, was released on May 10, 2007.

For more information, visit the Ubuntu Studio website.

Parrot Security OS

Parrot Security OS is a Linux distribution based on Debian with a focus on computer security. It is designed for penetration testing, vulnerability assessment, and mitigation, computer forensics, and anonymous web browsing.

For more information, visit the Parrot Security OS website.

Kali Linux

Kali Linux is a Debian-derived Linux distribution designed for digital forensics and penetration testing. It is maintained and funded by Offensive Security Ltd. It was developed by Mati Aharoni and Devon Kearns of Offensive Security through the rewrite of BackTrack, their previous forensics Linux distribution.

For more information, visit the Kali Linux website.

BlackArch Linux

BlackArch Linux is an Arch Linux-based distribution for penetration testers and security researchers. It is supplied as a live DVD image that comes with several lightweight window managers, including Fluxbox, Openbox, Awesome, and spectrwm.

For more information, visit the BlackArch Linux website.

ArchBang

ArchBang is a simple GNU/Linux distribution based on Archlinux system combined with the Openbox window manager. Suitable for both desktop and portable systems - It is fast, stable, and always up to date.

For more information, visit the ArchBang website.

Artix Linux

Artix Linux is a rolling-release distribution, based on Arch Linux. It uses OpenRC, runit, and s6 init instead of systemd. Artix Linux offers a lightweight, rolling-release operating system featuring the OpenRC init software.

For more information, visit the Artix Linux website.

EndeavourOS

EndeavourOS is a rolling-release Linux distribution based on Arch Linux. It comes with a friendly community and a helpful forum. The distribution is aimed at people who want a fast, stable, and clean system.

For more information, visit the EndeavourOS website.

Garuda Linux

Garuda Linux is a rolling-release Linux distribution based on Arch Linux. It comes with a friendly community and a helpful forum. The distribution is aimed at people who want a fast, stable, and clean system.

For more information, visit the Garuda Linux website.

Void Linux

Void Linux is an independent Linux distribution that uses the X Binary Package System (XBPS) package manager, which was designed and implemented from scratch, and the runit init system. The distribution is available for the Intel x86, ARM, and MIPS architectures with a focus on simplicity, clarity, and minimalism.

For more information, visit the Void Linux website.

Qubes OS

Qubes OS is a security-focused desktop operating system that aims to provide security through isolation. Virtualization is performed by Xen, and user environments can be based on Fedora, Debian, Whonix, and Microsoft Windows, among other operating systems.

For more information, visit the Qubes OS website.
