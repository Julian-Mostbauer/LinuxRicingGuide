Window Managers

Hyprland

Hyprland is a lightweight, highly configurable window manager for X11. It is designed to be fast and efficient, while still providing a high degree of customization. Hyprland is written in C and uses the Xlib library for communication with the X server.

For more information, visit the Hyprland website.

i3

i3 is a tiling window manager designed for X11. It is primarily targeted at advanced users and developers. i3 is written in C and uses the Xlib library for communication with the X server.

For more information, visit the i3 website.

Openbox

Openbox is a lightweight, highly configurable window manager for X11. It is designed to be fast and efficient, while still providing a high degree of customization. Openbox is written in C and uses the Xlib library for communication with the X server.

For more information, visit the Openbox website.

Awesome

Awesome is a highly configurable, tiling window manager for X11. It is designed to be fast and efficient, while still providing a high degree of customization. Awesome is written in C and uses the Xlib library for communication with the X server.

For more information, visit the Awesome website.

bspwm

bspwm is a tiling window manager that represents windows as the leaves of a full binary tree. It is designed to be fast and efficient, while still providing a high degree of customization. bspwm is written in C and uses the Xlib library for communication with the X server.
